Formulario interactivo en Vue 3

Este proyecto implementa una página de registro/perfil en Vue 3 que demuestra el uso de two-way binding con v-model, value binding con :value, validaciones básicas y actualización del estado del formulario en tiempo real.

El objetivo de la actividad es practicar cómo Vue sincroniza el estado de la aplicación con la interfaz de usuario utilizando sus directivas reactivas.

Tecnologías utilizadas

- Vue 3
- Vite
- JavaScript
- HTML5
- CSS3

Cómo correr el proyecto
1. Clonar el repositorio

2. Instalar dependencias

npm install

3. Ejecutar el servidor de desarrollo

npm run dev
